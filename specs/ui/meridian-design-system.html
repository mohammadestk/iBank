Meridian

Material 3 design system · Banking

A Material 3 implementation for a consumer banking app. Seed color is a deep emerald (#0B6E4F) rather than the default Material blue/purple — chosen for the dual association of trust (deep, desaturated) and growth (green), with a warm gold tertiary reserved for positive-money and highlight moments. Monetary figures are set in tabular-numeral mono throughout so amounts never jitter or misalign in lists.

What this file is, and isn't: a static token system + 4 mockups, not a working app. Three things to know before you build on it — (1) the tonal palettes here are hand-tuned hex steps for visual plausibility, not generated through Material's actual HCT color-space algorithm, so don't assume WCAG contrast holds at every pairing without re-checking, especially tertiary-on-white at tone 80/90; (2) there's no dark theme — M3 banking apps need one, and tonal surfaces invert non-trivially, it isn't just "flip the neutrals"; (3) no states are shown for error/empty/loading on Search or Messages, and there's no auth/onboarding flow, both of which are usually where banking app design actually gets hard. Treat this as a starting token set, not a spec ready for handoff.
